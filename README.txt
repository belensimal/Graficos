El proyecto está desarrollado en Java, usando Eclipse. La 
clase donde está el main es InterfazControl.java

El panel de control está dividido en 4:
	1- CREACIÓN DE FIGURAS
    Para crear una figura, pulse píxeles para ir creando 
    líneas. Cuando tenga la figura deseada, pulse el botón 
    "Nueva figura" y la figura se añadirá a la lista de 
    figuras.
	2-BORRAR Y ANIMACIÓN
    Para animar una figura, seleccionela de la lista y pulse
    en el botón "Animación". Para borrar toda la actividad 
    registrada, pulse en el botón "Borrar".
	3-TRANSFORMACIONES
    Selecciones la figura deseada a tranformar y cambie los
    valores de las variables. Cuando tenga la transformación 
    deseada, pulse en el botón "Aplicar" y la transformación 
    se dibujará en la pantalla.
	4-LISTA
    Lista de figuras creadas. Puedes seleccionarlas para 
    transformarlas.
